# Screenshots Directory

This directory contains screenshots and documentation images for the Grocery Price Scraper project.

## Screenshot Files

1. **Screencapture_features.png** - Overview of the scraper's key features and capabilities
2. **Screencapture-test.png** - Command-line output showing test execution and data collection process
3. **Screenshot_email.png** - Sample email report with summary statistics and Google Sheet link
4. **Screenshot_email_test.png** - Email notification test showing successful delivery
5. **Screenshot_sheet.png** - Google Sheets interface showing weekly data tabs and formatted product information
6. **Screenshot_total.png** - Summary statistics showing total products collected, stores processed, and data metrics

## Image Guidelines

- Use PNG format for screenshots
- Recommended resolution: 1920x1080 or higher
- Keep file sizes reasonable (< 2MB per image)
- Use descriptive filenames

## Adding New Screenshots

1. Take screenshots of relevant features or outputs
2. Save them in this directory with descriptive names
3. Update the main README.md to reference new screenshots
